Proof: Turning an Unbounded AR(1) Model to a Bounded One

(back to 03. ARMA Models)

(back to index)

We begin with an $AR(1)$ model:

\begin{equation*} x[t] = \phi x[t - 1] + w[t] \end{equation*}

where $|\phi| > 1$. Since $\phi$ will cause $x[t]$ to increase at each present value, the system is unbounded as time goes on. The value of $x[t]$ will approach infinity.

Let's write the future value $x[t + 1]$ as:

\begin{equation*} x[t + 1] = \phi x[t] + w[t + 1] \end{equation*}

Solving for $x[t]$ gives:

\begin{align*} x[t + 1] &= \phi x[t] + w[t + 1] \\ \phi x[t] &= x[t + 1] - w[t + 1]\\ x[t] &= \frac{1}{\phi}x[t + 1] - \frac{1}{\phi}w[t + 1] \\ \end{align*}

We can write the present value $x[t]$ in terms of future values:

\begin{equation*} x[t] = \frac{1}{\phi}x[t + 1] - \frac{1}{\phi}w[t + 1] \end{equation*}

Iterating forwards once (1st time) gives:

\begin{align*} x[t] &= \frac{1}{\phi}x[t + 1] - \frac{1}{\phi}w[t + 1] \\ &= \frac{1}{\phi} \left(\frac{1}{\phi}x[t + 2] - \frac{1}{\phi}w[t + 2] \right) - \frac{1}{\phi}w[t + 1] \\ &= \frac{1}{\phi^2}x[t + 2] - \frac{1}{\phi^2}w[t + 2] - \frac{1}{\phi}w[t + 1] \\ &= \frac{1}{\phi^2}x[t + 2] - \sum_{j = 1}^{2} \frac{1}{\phi^j} w[t + j] \end{align*}

Iterating forwards again (2nd time) gives:

\begin{align*} x[t] &= \frac{1}{\phi^2}x[t + 2] - \sum_{j = 1}^{2} \frac{1}{\phi^j} w[t + j] \\ &= \frac{1}{\phi^2} \left(\frac{1}{\phi}x[t + 3] - \frac{1}{\phi}w[t + 3] \right) - \sum_{j = 1}^{2} \frac{1}{\phi^j} w[t + j] \\ &= \frac{1}{\phi^3} x[t + 3] - \frac{1}{\phi^3}w[t + 3] - \sum_{j = 1}^{2} \frac{1}{\phi^j} w[t + j] \\ &= \frac{1}{\phi^3} x[t + 3] - \sum_{j = 1}^{3} \frac{1}{\phi^j} w[t + j] \end{align*}

Iterating forwards $k$ times gives:

\begin{equation*} x[t] = \frac{1}{\phi^{k + 1}} x[t + (k + 1)] - \sum_{j = 1}^{k + 1} \frac{1}{\phi^j} w[t + j] \end{equation*}

Since $|\frac{1}{\phi}| < 1$, writing $x[t]$ with future values bounds the time series. Extending this to numerous samples:

\begin{align*} x[t] &= \lim_{k \to \infty} \frac{1}{\phi^{k + 1}} x[t + (k + 1)] - \sum_{j = 1}^{k + 1} \frac{1}{\phi^j} w[t + j] \\ &= 0 - \sum_{j = 1}^{\infty} \frac{1}{\phi^j} w[t + j] \\ &= - \sum_{j = 1}^{\infty} \frac{1}{\phi^j} w[t + j] \\ \end{align*}

Thus a working stationary $AR(1)$ model can be created out of white noise future values:

\begin{equation*} x[t] = - \sum_{j = 1}^{\infty} \frac{1}{\phi^j} w[t + j] \end{equation*}
