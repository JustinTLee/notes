Proof: Generating an AR(1) Model out of White Noise Components

(back to 03. ARMA Models)

(back to index)

We begin with an $AR(1)$ model:

\begin{equation*} x[t] = \phi x[t - 1] + w[t] \end{equation*}

Iterating backwards once (1st time) gives:

\begin{align*} x[t] &= \phi x[t - 1] + w[t] \\ &= \phi (\phi x[t - 2] + w[t - 1]) + w[t] = \phi^2 x[t - 2] + \phi w[t - 1] + w[t] \\ &= \phi^2 x[t - 2] + \sum_{j = 0}^{1} \phi^j w[t - j] \end{align*}

Iterating backwards again (2nd time) gives:

\begin{align*} x[t] &= \phi^2 x[t - 2] + \sum_{j = 0}^{1} \phi^j w[t - j] \\ &= \phi^2 (\phi x[t - 3] + w[t - 2]) + \sum_{j = 0}^{1} \phi^j w[t - j] \\ &= \phi^3 x[t - 3] + \phi^2 w[t - 2] + \sum_{j = 0}^{1} \phi^j w[t - j] \\ &= \phi^3 x[t - 3] + \sum_{j = 0}^{2} \phi^j w[t - j] \end{align*}

Iterating backwards again (3rd time) gives:

\begin{align*} x[t] &= \phi^3 x[t - 3] + \sum_{j = 0}^{2} \phi^j w[t - j] \\ &= \phi^3 (\phi x[t - 4] + w[t - 3]) + \sum_{j = 0}^{2} \phi^j w[t - j] \\ &= \phi^4 x[t - 4] + \phi^3 w[t - 3] + \sum_{j = 0}^{2} \phi^j w[t - j] \\ &= \phi^4 x[t - 4] + \sum_{j = 0}^{3} \phi^j w[t - j] \end{align*}

Iterating backwards $k$ times gives:

\begin{equation*} x[t] = \phi^{k + 1} x[t - (k + 1)] + \sum_{j = 0}^{k} \phi^j w[t - j] \end{equation*}

If $|\phi| < 1$, then the equation gets reduced to:

\begin{align*} x[t] &= \lim_{k \to \infty} \phi^{k + 1} x[t - (k + 1)] + \sum_{j = 0}^{k} \phi^j w[t - j] \\ &= 0 + \sum_{j = 0}^{k} \phi^j w[t - j] \\ &= \sum_{j = 0}^{k} \phi^j w[t - j] \end{align*}

Thus, an $AR(1)$ model can be written as:

\begin{equation} x[t] = \sum_{j = 0}^{k} \phi^j w[t - j] \end{equation}
