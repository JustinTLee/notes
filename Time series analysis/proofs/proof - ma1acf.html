Proof: Proof: MA(1) Autocovariance

(back to 03. ARMA Models)

(back to index)

We begin with an $MA(1)$ model:

\begin{equation*} x[t] = w[t] + \theta w[t - 1] \end{equation*}

Sticking this equation into the autocovariance operator:

\begin{align*} \gamma[h] &= Cov\{x[t + h], x[t]\} \\ &= E\left\{w[t + h] + \theta w[t + h - 1], w[t] + \theta w[t - 1] \right\} \\ &= E\left\{(w[t + h] + \theta w[t + h - 1]) (w[t] + \theta w[t - 1]) \right\}\\ \end{align*}

Distributing the terms in the expectation operator:

\begin{align*} \gamma[h] &= E\left\{(w[t + h] + \theta w[t + h - 1]) (w[t] + \theta w[t - 1]) \right\} \\ &= E\left\{w[t + h]w[t] + \theta w[t + h]w[t - 1] + \theta w[t]w[t + h - 1] + \theta^2 w[t - 1]w[t + h - 1] \right\} \\ &= E\{w[t + h]w[t]\} + \theta E\{w[t + h]w[t - 1]\} + \theta E\{w[t]w[t + h - 1]\} + \theta^2 E\{w[t - 1]w[t + h - 1]\} \\ \end{align*}

Assume $h = 0$:

\begin{align*} \gamma[0] &= E\{w[t + 0]w[t]\} + \theta E\{w[t + 0]w[t - 1]\} + \theta E\{w[t]w[t + 0 - 1]\} + \theta^2 E\{w[t - 1]w[t + 0 - 1]\} \\ &= E\{w[t]w[t]\} + \theta E\{w[t]w[t - 1]\} + \theta E\{w[t]w[t - 1]\} + \theta^2 E\{w[t - 1]w[t - 1]\} \\ &= \sigma_w^2 + 0 + 0 + \theta^2 \sigma_w^2 \\ &= \sigma_w^2 + \theta^2 \sigma_w^2 = (\theta^2 + 1) \sigma_w^2 \\ \end{align*}

Now assume $h = 1$:

\begin{align*} \gamma[1]] &= E\{[t + 1]w[t]\} + \theta E\{w[t + 1]w[t - 1]\} + \theta E\{w[t]w[t + 1 - 1]\} + \theta^2 E\{w[t - 1]w[t + 1 - 1]\} \\ &= E\{[t + 1]w[t]\} + \theta E\{w[t + 1]w[t - 1]\} + \theta E\{w[t]w[t]\} + \theta^2 E\{w[t - 1]w[t]\} \\ &= 0 + 0 + \theta \sigma_w^2 + 0 \\ &= \theta \sigma_w^2 \end{align*}

Now assume $h = 2$:

\begin{align*} \gamma[2]] &= E\{[t + 2]w[t]\} + \theta E\{w[t + 2]w[t - 1]\} + \theta E\{w[t]w[t + 2 - 1]\} + \theta^2 E\{w[t - 1]w[t + 2 - 1]\} \\ &= E\{[t + 2]w[t]\} + \theta E\{w[t + 2]w[t - 1]\} + \theta E\{w[t]w[t + 1]\} + \theta^2 E\{w[t - 1]w[t + 1]\} \\ &= 0 + 0 + 0 + 0 \\ &= 0 \end{align*}

The $MA(1)$ model's autocovariance function is then:

\begin{align*} \gamma[h] &= (1 + \theta^2) \sigma_w^2 \quad &h = 0 \\ \gamma[h] &= \theta \sigma_w^2 \quad &h = 1 \\ \gamma[h] &= 0 \quad &h > 1 \\ \end{align*}

References:
