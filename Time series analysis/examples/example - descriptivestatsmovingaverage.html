Mean of a Moving Average Series:

This problem is taken from [1], example 1.13.

Suppose we wanted to to take this moving average:

\begin{equation*} E\{x[t]\} = E\left\{\frac{w[t - 1] + w[t] + w[t + 1]}{3}\right\} \end{equation*}

Remember, $E\{w[t]\} = 0$:

\begin{align*} E\{x[t]\} &= E\left\{\frac{w[t - 1] + w[t] + w[t + 1]}{3}\right\} \\ &= \frac{1}{3} E\{w[t - 1]\} + \frac{1}{3} E\{w[t]\} + \frac{1}{3} E\{w[t + 1]\} \\ &= \frac{1}{3} (0) + \frac{1}{3} (0) + \frac{1}{3} (0) \\ &= 0 \end{align*}

Autocovariance of a Moving Average Series:

This problem is taken from [1], example 1.17.

Suppose we wanted to to take this autocovariance:

\begin{align*} E\{x[t], x[t + h]\} &= E\left\{\frac{w[t - 1] + w[t] + w[t + 1]}{3}, \frac{w[t + h - 1] + w[t + h] + w[t + h + 1]}{3}\right\} \\ &= E\left\{\left(\frac{w[t - 1] + w[t] + w[t + 1]}{3}\right) \left(\frac{w[t + h - 1] + w[t + h] + w[t + h + 1]}{3}\right) \right\} \end{align*}

Remember, $E\{w[t], w[t + h]\} = 0$ where $h \ne 0$. Exapnding out the final expression above:

\begin{align*} &E\left\{\left(\frac{w[t - 1] + w[t] + w[t + 1]}{3}\right) \left(\frac{w[t + h - 1] + w[t + h] + w[t + h + 1]}{3}\right) \right\} \\ &= \frac{1}{9} E\left\{w[t - 1]w[t + h - 1] + w[t - 1]w[t + h] + w[t - 1]w[t + h + 1] + w[t]w[t + h - 1] + w[t]w[t + h] + w[t]w[t + h + 1] + w[t + 1]w[t + h - 1] + w[t + 1]w[t + h] + w[t + 1]w[t + h + 1] \right\} \end{align*}

Assume that $h = 0$:

\begin{align*} & \frac{1}{9} E\left\{w[t - 1]w[t + 0 - 1] + w[t - 1]w[t + 0] + w[t - 1]w[t + 0 + 1] + w[t]w[t + 0 - 1] + w[t]w[t + 0] + w[t]w[t + 0 + 1] + w[t + 1]w[t + 0 - 1] + w[t + 1]w[t + 0] + w[t + 1]w[t + 0 + 1] \right\} = \\ & \frac{1}{9} E\left\{w[t - 1]w[t - 1] + w[t - 1]w[t] + w[t - 1]w[t + 1] + w[t]w[t - 1] + w[t]w[t] + w[t]w[t + 1] + w[t + 1]w[t - 1] + w[t + 1]w[t] + w[t + 1]w[t + 1] \right\} = \\ & \frac{1}{9} \left(E\left\{w[t - 1]w[t - 1]\right\} + E\left\{w[t - 1]w[t]\right\} + E\left\{w[t - 1]w[t + 1]\right\} + E\left\{w[t]w[t - 1]\right\} + E\left\{w[t]w[t]\right\} + E\left\{w[t]w[t + 1]\right\} + E\left\{w[t + 1]w[t - 1]\right\} + E\left\{w[t + 1]w[t]\right\} + E\left\{w[t + 1]w[t + 1]\right\}\right) = \\ & \frac{1}{9} \left(\sigma_w^2 + 0 + 0 + 0 + \sigma_w^2 + 0 + 0 + 0 + \sigma_w^2\right) = \\ & \frac{\sigma_w^2 + \sigma_w^2 + \sigma_w^2}{9} = \frac{3 \sigma_w^2}{9} = \\ & \frac{\sigma_w^2}{3} \end{align*}

Assume that $h = 1$:

\begin{align*} & \frac{1}{9} E\left\{w[t - 1]w[t + 1 - 1] + w[t - 1]w[t + 1] + w[t - 1]w[t + 1 + 1] + w[t]w[t + 1 - 1] + w[t]w[t + 1] + w[t]w[t + 1 + 1] + w[t + 1]w[t + 1 - 1] + w[t + 1]w[t + 1] + w[t + 1]w[t + 1 + 1] \right\} = \\ & \frac{1}{9} E\left\{w[t - 1]w[t] + w[t - 1]w[t + 1] + w[t - 1]w[t + 2] + w[t]w[t] + w[t]w[t + 1] + w[t]w[t + 2] + w[t + 1]w[t] + w[t + 1]w[t + 1] + w[t + 1]w[t + 2] \right\} = \\ & \frac{1}{9} \left(E\left\{w[t - 1]w[t]\right\} + E\left\{w[t - 1]w[t + 1]\right\} + E\left\{w[t - 1]w[t + 2]\right\} + E\left\{w[t]w[t]\right\} + E\left\{w[t]w[t + 1]\right\} + E\left\{w[t]w[t + 2]\right\} + E\left\{w[t + 1]w[t]\right\} + E\left\{w[t + 1]w[t + 1]\right\} + E\left\{w[t + 1]w[t + 2]\right\}\right) = \\ & \frac{1}{9} \left(0 + 0 + 0 + \sigma_w^2 + 0 + 0 + 0 + \sigma_w^2 + 0 \right) = \\ & \frac{\sigma_w^2 + \sigma_w^2}{9} = \\ & \frac{2 \sigma_w^2}{9} \end{align*}

Assume that $h = 2$:

\begin{align*} & \frac{1}{9} E\left\{w[t - 1]w[t + 2 - 1] + w[t - 1]w[t + 2] + w[t - 1]w[t + 2 + 1] + w[t]w[t + 2 - 1] + w[t]w[t + 2] + w[t]w[t + 2 + 1] + w[t + 1]w[t + 2 - 1] + w[t + 1]w[t + 2] + w[t + 1]w[t + 2 + 1] \right\} = \\ & \frac{1}{9} E\left\{w[t - 1]w[t + 1] + w[t - 1]w[t + 2] + w[t - 1]w[t + 3] + w[t]w[t + 1] + w[t]w[t + 2] + w[t]w[t + 3] + w[t + 1]w[t + 1] + w[t + 1]w[t + 2] + w[t + 1]w[t + 3] \right\} = \\ & \frac{1}{9} \left(E\left\{w[t - 1]w[t + 1]\right\} + E\left\{w[t - 1]w[t + 2]\right\} + E\left\{w[t - 1]w[t + 3]\right\} + E\left\{w[t]w[t + 1]\right\} + E\left\{w[t]w[t + 2]\right\} + E\left\{w[t]w[t + 3]\right\} + E\left\{w[t + 1]w[t + 1]\right\} + E\left\{w[t + 1]w[t + 2]\right\} + E\left\{w[t + 1]w[t + 3]\right\}\right) = \\ & \frac{1}{9} \left(0 + 0 + 0 + 0 + 0 + 0 + \sigma_w^2 + 0 + 0 \right) = \\ & \frac{\sigma_w^2}{9} \\ \end{align*}

Assume that $h = 3$:

\begin{align*} & \frac{1}{9} E\left\{w[t - 1]w[t + 3 - 1] + w[t - 1]w[t + 3] + w[t - 1]w[t + 3 + 1] + w[t]w[t + 3 - 1] + w[t]w[t + 3] + w[t]w[t + 3 + 1] + w[t + 1]w[t + 3 - 1] + w[t + 1]w[t + 3] + w[t + 1]w[t + 3 + 1] \right\} = \\ & \frac{1}{9} E\left\{w[t - 1]w[t + 2] + w[t - 1]w[t + 3] + w[t - 1]w[t + 4] + w[t]w[t + 2] + w[t]w[t + 3] + w[t]w[t + 4] + w[t + 1]w[t + 2] + w[t + 1]w[t + 3] + w[t + 1]w[t + 4] \right\} = \\ & \frac{1}{9} \left(E\left\{w[t - 1]w[t + 2]\right\} + E\left\{w[t - 1]w[t + 3]\right\} + E\left\{w[t - 1]w[t + 4]\right\} + E\left\{w[t]w[t + 2]\right\} + E\left\{w[t]w[t + 3]\right\} + E\left\{w[t]w[t + 4]\right\} + E\left\{w[t + 1]w[t + 2]\right\} + E\left\{w[t + 1]w[t + 3]\right\} + E\left\{w[t + 1]w[t + 4]\right\}\right) = \\ & \frac{1}{9} \left(0 + 0 + 0 + 0 + 0 + 0 + 0 + 0 + 0 \right) = \\ & 0 \\ \end{align*}

$\gamma[h] = 0$ for $h \ge 3$. Therefore, the autocovariance of this particular moving average series becomes:

\begin{align*} \gamma[0] &= \frac{3 \sigma_w^2}{9} = \frac{\sigma_w^2}{3} \\ \gamma[1] &= \frac{2 \sigma_w^2}{9} \\ \gamma[2] &= \frac{\sigma_w^2}{9} \\ \gamma[h \ge 3] &= 0 \\ \end{align*}
