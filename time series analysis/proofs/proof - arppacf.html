We begin with the following definition:

\begin{equation*} \widehat{x}[t + h] = \sum_{j = 1}^{h - 1} \beta_j x[t + h - j] \end{equation*}

If $h > p$, we can expand out our previous summation into two summations:

\begin{align*} \widehat{x}[t + h] &= \sum_{j = 1}^{h - 1} \beta_j x[t + h - j] \\ &= \sum_{j = 1}^{p} \beta_j x[t + h - j] + \sum_{j = p + 1}^{h - 1} \beta_j x[t + h - j] \end{align*}

Given that an $AR(p)$ process can be written as $x[t + h] = \sum_{j = 1}^{p} \phi_{j} x[t + h - m] + w[t + h]$, it can assumed that in order to mimic this form, we can set $\sum_{j = p + 1}^{h - 1} \beta_j x[t + h - j] = 0$:

\begin{align*} \widehat{x}[t + h] &= \sum_{j = 1}^{p} \beta_j x[t + h - j] + \sum_{j = p + 1}^{h - 1} \beta_j x[t + h - j] \\ &= \sum_{j = 1}^{p} \beta_j x[t + h - j] + 0 \\ &= \sum_{j = 1}^{p} \beta_j x[t + h - j] \\ \end{align*}
